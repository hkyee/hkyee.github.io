Behavior-in-the-loop testing of Automated Driving System (ADS) for virtual safety testing

Linux IPG CarMaker Baidu Apollo Python Docker

Final Year Project:

  • Investigated the capability of Baidu Apollo, an existing open-source platform for Autonomous Driving System (ADS) model.
  • Deep analysis on Baidu Apollo’s modules including Perception, Prediction, Planning, Localization and Control.
  • Integrated the ADS model with IPG CarMaker, a virtual simulation tool for scenario-based testing, using IPG CarMaker Python API.
  • Implemented client-server model using TCP/IP for establishing communication between IPG CarMaker and Baidu Apollo.
  • Utilized Cyber RT framework, which follows the concept of Robotic Operating Systems (ROS).

Temperature Measurement System with Built-in Memory

VHDL Field-Programmable Gate Arrays (FPGA) Xilinx ISE

Project:

  • Designed a real-time temperature measurement system with Spartan-3E (Xilinx XC3S500E FPGA)
  • Features include:
    • SPI for data communication with an external temperature sensor
    • Able to display temperature on the six digit 7-segment display and LCD screen.
    • Dual Port RAM integration to store the last measured temperature value.
    • RS232 as serial communication for displaying temperatures on a desktop terminal.
